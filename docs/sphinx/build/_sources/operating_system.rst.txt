***********
OSUtilities
***********

The ``OSUtilities`` class is stored in the ``operating_system`` package.  This 
class contains many functions that allow a user to access comman line features
from the Python console or from a Python script encoded to a text file.  This class is 
written with class methods that simplify or augment many of the utilities offered in
`os <https://docs.python.org/3/library/os.html>`_ and `sys <https://docs.python.org/3/library/sys.html>`_ packages
