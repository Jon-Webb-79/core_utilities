OSUtilities¶

The OSUtilities class is stored in the operating_system package. This class contains many functions that allow a user to access comman line features from the Python console or from a Python script encoded to a text file. This class is written with class methods that simplify or augment many of the utilities offered in os and sys packages

class operating_system.OSUtilities[source]¶

This set of functions were written as class methods in order to ensure that they can be passed in bulk via decorator patters or through inheritance. This class provides file analysis capabilities and a user interface for file and directory management similar to Bash and DOS functionality, although with different syntax

All code examples are predicated on the instantiation of this class with the following command

> util = OSUtilities()
classmethod change_directory(new_directory: str) → None[source]¶
Parameters

new_directory – The desired directory to include path link

Return None

This function will change the directory in the same manner as the cd DOS and Linux command. For example, if you are in the dir1 directory within the following chain;

home/dir1/dir2/

You can use the following python script to change to the dir2 directory.

> print(util.current_working_directory())
'/home/dir1'
> util.change_directory('dir2')
> print(util.current_working_directory())
'/home/dir1/dir2'

You can change back to the home directory with the following command;

> util.change_directory('../..')
> print(util.current_working_directory())
'/home'
classmethod copy_directory(source: str, destination: str) → None[source]¶
Parameters

  • source – The name and path-link of the directory to be copied

  • destination – The name and path-link for the directory copy

Return None

This function creates a copy of a directory and assigns it to the directory of the users choosing. This function will copy populated and un-populated directories. As an example lets assume the following directory structure;

directory_1
   |
   text_file.txt
   directory_2
   directory_3
      |
      another_text_file.txt

The following command will make copy of directory_2 in the directory_3 titled dir2_copy, directory assuming we are in the directory_1 directory

> copy_directory('directory_2', 'directory_3/dir2_copy')
> change_directory('directory_3')
> print(list_contents())
[''another_text_file.txt', 'dir2_copy']

classmethod count_occurrence_of_word_in_file(file_name: str, word: str) → int[source]¶
Parameters

  • file_name – The file name to include the path link

  • word – the word for which the number of occurrences in a file is desired.

Return num

The number of times a specific word occurs in a file.

This function excludes the effects of punctuation. For example if the user defined word is book and the file contains the word book. with a period following it indicating the end of a sentence, this function will count that word. The same is true for commas. As an example lets assume the file listed below, titled test.txt;

This is a text file, all
of the information in this
file is specifically for
the purpose of testing file file

The following code will count the occurrence of the word file

> num = util.count_occurrence_of_word_in_file('test.txt', 'file')
> print(num)
4

   text_file.txt
